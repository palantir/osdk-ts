# Example

The starter component shipped with `@monorepo-app-placeholder/components`. Edit `packages/components/src/components/Example.tsx`.

<ComponentPage packageName="@monorepo-app-placeholder/components" props={[
  { name: "title", type: "string", required: true, description: "Title shown at the top of the card." },
  { name: "icon", type: "ReactNode", description: "Optional React node rendered next to the title (Blueprint icon, SVG, emoji, anything)." },
  { name: "description", type: "string", required: true, description: "Body text." },
]}>
