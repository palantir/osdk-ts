# Blueprint components

[Blueprint](https://blueprintjs.com/) is the React component library built by Palantir.

---

It offers an extensive set of components optimized for building complex, data-dense interfaces for desktop applications — tables, forms, dialogs, menus, date pickers, and more — all with consistent styling, full keyboard support, and a comprehensive icon set. In this template, Blueprint powers the docs site itself — the tab bar at the top of each component page, the `<Playground>` controls (segmented controls, switches, sliders, selects), and the icon picker in the starter `Example` are all Blueprint components.
